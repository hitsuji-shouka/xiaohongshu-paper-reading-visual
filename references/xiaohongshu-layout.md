# 小红书 3:4 排版规范

## 画布

```text
比例：3:4 竖版
推荐尺寸：1080 x 1440
安全边距：左右 80-100 px，顶部 80-100 px，底部 100-120 px
```

未经用户明确要求，不要回退到 16:9。

## 区域建议

```text
顶部 15%-20%：
页码、主标题、副标题

中部 55%-65%：
主插画、角色动作、技术隐喻

底部 20%-25%：
简短说明、公式、命令、结论或速查信息
```

## 文字策略

- 标题必须适合手机阅读。
- 每页正文尽量 1-3 条，不写长段落。
- 代码、命令、公式、快捷键优先后期排版添加。
- 生图内短标签控制在 2-8 个汉字。
- 不让文字贴边、遮挡角色或挤压主视觉。

## 交付要求

- 最终轮播应为独立页面，而不是只有总览拼图。
- 总览图只能作为额外预览。
- 所有页面保持统一页码、标题层级、留白、线条和强调色。
