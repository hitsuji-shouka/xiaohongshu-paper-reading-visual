# 署名规范

本项目的共享视觉方法参考并改编自 Ian 创建的开源项目：

- 项目名称：Ian Xiaohei Illustrations
- GitHub：`helloianneo/ian-xiaohei-illustrations`

公开发布、改编或再分发本项目时：

1. 保留根目录 `README.md` 中的 Attribution 章节。
2. 保留根目录 `NOTICE.md`。
3. 不要声称拥有 Ian 原始视觉系统、原始角色 IP 或原始示例图片。
4. 示例图优先使用本项目新生成的内容，不直接复制原仓库示例图。
5. 若继续衍生新的视觉角色，明确说明它是独立角色，不是 Ian 原始角色。
