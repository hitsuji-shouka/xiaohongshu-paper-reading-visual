# 小红书论文分享模板

## 快速分享：1-3 页

适合用户说“先做个演示”“想快速分享这篇文章”“给我一张总结图”。

```text
第 1 页：封面 / 核心概念
第 2 页：Paper Map 总结构
第 3 页：我的启发 / 风险提醒（可选）
```

## 完整解读：6-8 页

适合用户要完整小红书轮播。

```text
1. 封面：论文名 + 最大冲突
2. 一句话总结：这篇论文讲什么
3. 解决的问题：为什么需要它
4. 核心方法：作者怎么做
5. 关键结果：证据是什么
6. 最大创新：真正新的视角
7. 主要局限：哪里不能过度解读
8. 启发：对研究/工程/产品有什么用
```

## 每页字段

```text
页码：
页面目的：
主标题：
一句话结论：
视觉结构：
推荐短标签：
必须准确的论文事实：
可放入生图的文字：
必须后期 overlay 的文字：
生图提示词：
```

## 标题方向

- `一张图看懂 XXX`
- `这篇论文真正想说什么？`
- `XXX：不是更强模型，而是新的工作方式`
- `读完这篇，我对 XXX 的理解变了`
- `这篇论文最值得记住的不是方法名`

## 发布文案

正文建议结构：

```text
最近读到一篇关于 XXX 的文章。
它最有意思的地方不是 XXX，而是 XXX。

我用 7 个问题拆了一遍：
1. 它解决什么问题
2. 方法是什么
3. 结果说明什么
4. 创新在哪里
5. 局限在哪里
6. 对我有什么启发

适合对 XXX 感兴趣的同学收藏。
```
